# あおば税理士事務所 コーポレートサイト

架空の税理士事務所「あおば税理士事務所」のポートフォリオ用Webサイトです。
信頼感と集客力を両立させたデザインと、高速なパフォーマンスを実現しています。

## サイトの特徴

*   **ターゲット**: 従業員5名以下の小規模法人、起業準備中の個人
*   **デザイン**: ネイビーとゴールドを基調とした、知的で信頼感のある配色
*   **技術**: HTML5, CSS3, Vanilla JavaScript (フレームワーク不使用)
*   **レスポンシブ**: スマートフォン、タブレット、PCに完全対応

## フォルダ構成

```text
aoba-tax-office/
├── index.html       # トップページ
├── service.html     # サービス・料金
├── about.html       # 代表プロフィール・事務所概要

├── css/
│   ├── globals.css  # 共通スタイル・変数定義
│   └── style.css    # ページ固有スタイル（必要に応じて記述）
└── js/
    └── main.js      # 共通スクリプト（メニュー、アニメーション）
```

## 編集方法

### 色やフォントの変更
`css/globals.css` の `:root` 内の変数を変更するだけで、サイト全体の色やフォントを一括変更できます。

```css
:root {
  --color-main: #1B3A5C;   /* メインカラー */
  --color-accent: #C4982F; /* アクセントカラー */
  ...
}
```

### 画像の差し替え
実際の運用時は、`https://placehold.co/...` の部分を実画像のパス（例: `images/hero.jpg`）に書き換えてください。

## 公開手順 (GitHub Pages)

1. リポジトリを作成し、すべてのファイルをアップロードします。
2. リポジトリの `Settings` > `Pages` を開きます。
3. `Source` を `Deploy from a branch` に設定します。
4. `Branch` を `main` (または `master`) / `/(root)` に設定し、Saveを押します。
5. 数分後、生成されたURLにアクセスすればサイトが公開されています。
